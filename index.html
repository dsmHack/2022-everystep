Create Cheer Box Labels

This webpage generates printable QR code labels for Cheer Boxes. Box details can be provided by exporting the "cheerbox_master" spreadsheet or by copying and pasting specific rows. For the scannable QR code, the Google Form URL may be specified to prefix each box's unique code.

Exporting from Excel
  1. Open the "cheerbox_master" Excel spreadsheet
  2. Navigate to "File" > "Save As" > "Download a Copy"
  3. Click the file upload below and select the just-downloaded ".xlsx" file
Copy/pasting from Excel
  1. Open the "cheerbox_master" spreadsheet and "qr_export" sheet
  2. Select one or more rows in the spreadsheet and copy them with CTRL+C
  3. Click anywhere on this webpage and paste with CTRL+V
